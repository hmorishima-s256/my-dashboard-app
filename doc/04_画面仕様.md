# 04. 画面仕様

## 4.1 メイン画面レイアウト

上部は 3 カラム構成です。

- 左: 日付表示ボタン + 日付入力パネル
- 中央: タブ (`予定表` / `タスク`)
- 右: `同期` ボタン + 認証UI（ログインボタンまたはプロフィールアイコン）

中央下部にメインコンテンツがあり、左下に設定ボタン、右下に更新日時を表示します。

## 4.2 ヘッダー部（左）

- 通常表示: `yyyy/mm/dd`
- ホバー時: 背景/枠/文字の見た目変化
- クリック時: 日付入力パネルをトグル表示

### 日付入力パネル

- 入力欄
  - `yyyy`（4桁）
  - `mm`（2桁）
  - `dd`（2桁）
- ボタン
  - `キャンセル`
  - `OK`
- キー操作
  - `dd` 入力欄で Enter -> OK と同等
- エラー表示
  - 入力欄赤枠
  - パネル内エラーメッセージ

## 4.3 ヘッダー部（中央）

- タブ: `予定表`, `タスク`
- アクティブタブはアクセント色（緑系グラデーション）
- 日付ボタン/同期ボタン/プロフィールアイコンと高さを揃える

## 4.4 ヘッダー部（右）

### 未ログイン

- `同期`（無効）
- `Googleログイン`

### ログイン済み

- `同期`（有効）
- 丸形プロフィールアイコン（固定アイコン）
- アイコン押下時メニュー
  - アイコン
  - 名前
  - メールアドレス
  - `ログアウト` ボタン

## 4.5 予定表タブ

- テーブル列
  - 日時
  - 件名
  - カレンダー名
- 日時セルは改行表示に対応（開始/終了 2行）
- ヘッダは sticky 表示

## 4.6 タスクタブ

- 現在はモック実装
- 選択日連動の文字列タスクをリスト表示

## 4.7 設定モーダル

- 開閉
  - 開く: 左下歯車
  - 閉じる: 背景クリック or `閉じる`
- 項目
  - 自動取得時刻
  - 自動取得間隔（値 + 単位）
- ボタン
  - `閉じる`
  - `未設定にする`
  - `保存`

## 4.8 ログアウト確認モーダル

- 文言: ログアウト可否確認
- ボタン
  - `キャンセル`
  - `ログアウト`

## 4.9 補助表示

- 右下: `更新日時: yyyy/mm/dd HH:mm:ss`
- レスポンシブ調整
  - 720px 以下で上部レイアウトを縦積みに変更
  - ボタン高さ/文字サイズを縮小
