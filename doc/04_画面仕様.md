# 04. 画面仕様

## 4.1 全体レイアウト

- ダークテーマのダッシュボード UI
- 上部: 3エリア構成
  - 左: 日付表示 + 年月日入力パネル
  - 中央: タブ (`予定表` / `タスク`)
  - 右: `同期` ボタン + 認証 UI
- 中央: コンテンツパネル（予定表またはタスク）
- 左下: 設定ボタン（歯車）
- 右下: 更新日時

## 4.2 ヘッダー左（日付）

- 通常表示: `yyyy/mm/dd`
- クリックで日付入力パネル表示
- ホバー時に背景と枠色を変化

### 日付入力パネル

- 入力欄: `yyyy` / `mm` / `dd`
- 操作ボタン: `キャンセル` / `OK`
- Enter 確定: `dd` 入力欄で Enter -> `OK` と同じ
- バリデーションエラー:
  - 入力欄を赤枠表示
  - エラーメッセージ表示

## 4.3 ヘッダー中央（タブ）

- タブ: `予定表` / `タスク`
- アクティブ時は緑系アクセントで強調
- 非アクティブ時は半透明背景
- 日付ボタン / 同期ボタン / アイコンボタンと高さを統一

## 4.4 ヘッダー右（同期 + 認証）

### 未ログイン

- `同期` ボタン（無効）
- `Googleログイン` ボタン

### ログイン済み

- `同期` ボタン（有効）
- 固定人型アイコン（丸形）
- アイコン押下でプロフィールメニュー表示
  - 名前
  - メール
  - `ログアウト`

## 4.5 予定表タブ

- テーブル列順:
  - `日時`
  - `件名`
  - `カレンダー名`
- 日時セル:
  - 終日: `1day`
  - 時刻予定: 開始/終了を2行表示
- テーブルヘッダーは sticky
- コンテンツは上寄せ表示

## 4.6 タスクタブ

- 上段: タスク登録フォーム
  - 案件（必須）/カテゴリ（任意）を候補選択または新規作成
  - タスク名（必須）
  - 優先度、初期ステータス、見積時刻、見積分、備考
- 下段: タスク一覧テーブル
  - 案件/カテゴリ
  - タスク
  - ステータス
  - 優先度
  - 見積
  - 実績
  - 操作（開始/中断/停止/完了/削除）
- ステータス色分け
  - `todo`: 黄系
  - `doing`: 青系
  - `suspend`: 橙系
  - `done`: 緑系
- ゲスト時は「終了時に消える」表示をフォーム上に表示

## 4.7 設定モーダル

- 開く: 左下歯車
- 閉じる:
  - 背景クリック
  - `閉じる`
  - `Esc`
- 項目:
  - 自動取得時刻
  - 自動取得間隔（値 + 単位）
- アクション:
  - `閉じる`
  - `未設定にする`
  - `保存`

## 4.8 ログアウト確認モーダル

- 文言: ログアウト確認
- ボタン:
  - `キャンセル`
  - `ログアウト`

## 4.9 補助表示

- 右下: `更新日時: yyyy/mm/dd HH:mm:ss`
- レスポンシブ:
  - 小画面で上部エリアを縦方向に再配置
